# Starter for site with admin panel on Laravel

- [Laravel 8](https://laravel.com/docs/8.x)
- [Bootstrap 5](https://getbootstrap.com/docs/5.0/getting-started/introduction/)

## Running
- `composer install`
- `php artisan serve`
- `php artisan migrate`
- `php artisan db:seed`
- `npm install` 
- `npm run production`
